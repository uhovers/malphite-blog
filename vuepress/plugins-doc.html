    VuePress 用到的一些插件

    Malphite 2022-4-14 vuepressjsplugin前端

    # 背景

    为了方便使用，VuePress 提供了一些插件，可以让你的文档更加美观。

    插件配置参考 (opens new window)

    # 使用插件的方法：

    你可以通过在.vuepress/config.js文件中的plugins数组中添加插件来实现。

    module.exports = {
    plugins: [
        require('./my-plugin'),
    ]
}

    # 常用插件

    # 公告栏弹窗 (opens new window)

    安装方法：

    npm install @vuepress-reco/vuepress-plugin-bulletin-popover -D
    1

    使用方法：

    plugins: [
    ['@vuepress-reco/vuepress-plugin-bulletin-popover', {
        width: '300px',   // 弹窗宽度 默认 260px
        title: '公告栏',
        body: [
            {
                type: 'title',
                content: '这里是公告栏的内容',
                style: 'text-align: center;'
            },
            {
                type: 'image',
                src: 'https://vuepress-theme-reco.recoluan.com/icon_vuepress_reco.png',
            }
        ],
        footer: [
            {
                type: 'button',
                text: '查看详情',
                link: '/donate'
            }
        ]
    }]
]

    # 搜索插件 (opens new window)

    安装方法：

    npm install @vuepress/plugin-search -D
    1

    使用方法：

    plugins: [
    ['@vuepress/search', {
        searchMaxSuggestions: 10    // 搜索的最大结果数 默认 5
    }]
]

    # 添加著作权信息 (opens new window)

    安装方法：

    npm install vuepress-plugin-copyright -D
    1

    使用方法：

        authorName: 'Malphite',

    # 背景音乐 (opens new window)

    安装方法：

    npm install @vuepress-reco/vuepress-plugin-bgm-player -D
    1

    使用方法：

    plugins: [
    ['@vuepress-reco/vuepress-plugin-bgm-player', { // 音乐播放器
        audios: [
            {
                name: '郭源潮', // 音乐名称
                artist: '宋东野',   // 艺术家
                url: 'https://www.ytmp3.cn/down/52237.mp3', // 音乐地址
                cover: 'https://p1.music.126.net/6y-UleORITEDbvrOLV0Q8A==/5639395138885805.jpg?param=130y130' // 封面图
            }
        ],
        autoplay: true, // 是否自动播放
        autoShrink: true ,  // 是否默认缩小
        shrinkMode: 'float',  // 缩小时缩为哪种模式
        floatStyle:{ bottom: '10px', 'z-index': '999999' }  // 悬浮窗样式
    }],
]

    安装方法：

    shape: 'star', // ['star' | 'circle'], // shape of the particle, default: 'star'
    zIndex: 999999999, // z-index property of the canvas, default: 999999999

    # 代码分离展示 (opens new window)

    安装方法：

    npm install @vuepress-reco/vuepress-plugin-extract-code -D
    1

    使用方法：

    plugins: [
    '@vuepress-reco/extract-code'
]
    1
    2
    3

    当你需要使用这种方式展示代码时,你需要在md文件添加：

    <RecoDemo :collapse="true">
    <template slot="code-bash">
        <<< @/docs/leetcode/code/index.sh
    </template>
    <template slot="code-js">
        <<< @/docs/leetcode/code/index.js
    </template>
</RecoDemo>

    3

    其中的@表示当前项目目录的别名,即源码的根目录,而非docs或者.vuepress

    @/docs/leetcode/code/index.sh和@/docs/leetcode/code/index.js是你的代码文件的路径
